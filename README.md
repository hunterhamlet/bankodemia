# Widgets guidelines

Cuando programemos un nuevo widget (UIView), podemos seguir los siguientes prefijos: 

* btn o button
* l o label
* tv o textView
* tbl o table
* cll o collection
* v o view
* ti o textInput
* iv o imageView
